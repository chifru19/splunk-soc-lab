# Sentinel Fortress | Splunk SIEM & SOC Lab
**Engineer:** Frank Fru
**Compliance:** NIST SP 800-61 (Incident Handling)

## Overview
This laboratory environment is a managed SIEM instance designed to monitor the digital infrastructure of Fon Ngwashi Law and Agwo Carlson & Partners. 

## Technical Stack
* **SIEM:** Splunk Enterprise (M1/M2 Apple Silicon Optimized)
* **Automation:** GitHub Actions for CI/CD Security Linting
* **Log Logic:** Apache Access Log Correlation (SPL)

## How to Access
1. Start the container: `docker start splunk-sentinel`
2. Search UI: `http://localhost:8000`
3. Credentials: `admin / SplunkSOC2026`
